Header
Main
Sidebar
Footer
.grid-template-areas {
   grid-template-columns: repeat(4, 1fr);
   /* . = empty area */
   grid-template-areas: 
     'header header  header  header'
     'main   main   .       sidebar'
     'footer footer  footer  footer'
 }
 .header { grid-area: header; }
 .main { grid-area: main; }
 .sidebar { grid-area: sidebar; }
 .footer { grid-area: footer; }

Grid Auto Flow, Auto Columns & Auto Rows

Control the auto-placement algorithm
Specifies the sizes of IMPLICIT grid tracks
grid-auto-flow | grid-auto-columns | grid-auto-rows

1
2
3
4
5
.auto-columns {
  grid-auto-flow: column; /* controls how the auto-placement algorithm works - row | column | dense */
  grid-template-columns: 50px;
  grid-auto-columns: 50px;
}
1
2
3
4
5
.auto-rows {
  grid-auto-flow: row; /* controls how the auto-placement algorithm works - row | column | dense */
  grid-template-columns: repeat(3, 1fr);     
  grid-auto-rows: 50px;
}
